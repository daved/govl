v(erify) l(inks)
===============

Go port (WIP) of python's [vl](https://github.com/ellisonleao/vl) lib

Install:

```
go get github.com/ellisonleao/govl
```

Usage:

```bash
  vl [flags] DOC

Flags:
  -d, --debug         Debug mode
  -t, --timeout int   HTTP Request Timeout (seconds) (default 5)
```
